# 📌 Overview

AI Chatbot is a modern desktop assistant developed using **C++**, **Qt Framework**, and **Object-Oriented Programming**.

It provides intelligent AI conversations through REST APIs while offering a beautiful desktop interface with authentication, voice support, chat history, and modern UI.

---

# ✨ Features

- 🤖 AI Chat Assistant
- 🔐 User Authentication
- 🎤 Voice Input
- 🔊 Text-to-Speech
- 💬 Real-time Conversations
- 🗂 Chat History
- 📎 File Attachments
- 😀 Emoji Support
- 🌙 Dark / Light Theme
- 👤 User Profiles
- 🔑 SHA-256 Password Hashing
- ⚡ Multiple AI Models
- 💾 Local Storage
- 🌐 REST API Integration
